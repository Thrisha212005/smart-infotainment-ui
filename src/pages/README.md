# 🧠 Smart Infotainment System (Voice + Gesture Controlled UI)

## 📌 Overview
A modern, modular infotainment system simulator built using React and TypeScript that demonstrates next-generation human–computer interaction. The system integrates voice recognition and gesture-based controls to create an immersive and interactive user experience.

It is designed as a multi-panel dashboard simulating real-world infotainment environments such as vehicle controls, music playback, navigation, phone integration, and climate control.

---

## ✨ Key Highlights
- 🎙 Voice-controlled interaction system for hands-free navigation  
- ✋ Gesture-based controls for intuitive UI interaction  
- 🎵 Integrated music player with local audio support  
- 🚗 Multi-panel infotainment dashboard simulation  
- 📱 Phone, navigation, climate, and music control modules  
- ⚡ Real-time state synchronization across components  
- 📊 Clean modular architecture using React Context API  

---

## 🧠 Core Features

### 🎧 Music System
- Play local audio tracks (`.wav`)
- Control playback (play/pause/next)
- Smooth UI feedback and state sync

### 🎙 Voice Control
- Voice command recognition using Web Speech API
- Hands-free interaction with system panels

### ✋ Gesture Control
- Mouse/touch-based gesture interactions
- Intuitive navigation across dashboard modules

### 🚗 Infotainment Dashboard
- Multi-module system:
  - Music Panel
  - Navigation Panel
  - Phone Panel
  - Climate Panel
  - Vehicle Panel

### 🧩 System Architecture
- Context-driven global state management
- Component-based modular design
- Real-time UI updates across panels

---

## 🛠 Tech Stack

- React (TypeScript)
- Vite (Build Tool)
- Tailwind CSS
- shadcn/ui component library
- HTML5 Audio API (music playback)
- Web Speech API (voice recognition)
- DOM Events API (gesture & interaction handling)
- React Context API (state management)

---

## 📁 Project Structure

- `/src/components/panels` → Infotainment modules  
- `/src/components/ui` → Reusable UI components (shadcn)  
- `/src/contexts` → Global state management  
- `/src/hooks` → Custom React hooks  
- `/public/music` → Audio assets  
- `/src/pages` → Application routes
